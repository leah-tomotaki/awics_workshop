# awics_workshop
 A GitHub workshop repo

 To participate:
 - Clone the repo
 - Add a card by copying the html and changing the text, image, and link.
 - Commit your changes
 - Push to the repo!

To test out branches:
- Create a new branch
- Change your image (or add a new card)
- Make a pull request
- Assign a reviewer (if applicable)
- Merge your changes!
